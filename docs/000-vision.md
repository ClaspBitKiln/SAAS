# 000 — Vision

## Целевой продукт
**AI Sales OS** — простая, надёжная и быстрая система для B2B-продаж в России и СНГ.
Для компаний 1–100 сотрудников.

Формула:
```
Communications + CRM + Telephony + Automation + AI + Counterparty Check
```

## Для кого
Менеджеры по продажам и руководители отделов продаж в B2B. Не корпорации, а МСБ,
которым Битрикс24 и amoCRM либо слишком тяжёлые, либо слишком дорогие.

## Чем отличаемся
- **CRM — ядро, а не дополнение.** Всё строится вокруг сделки и клиента.
- **Inbox-first.** Все каналы общения в одном месте, с историей и таймлайном.
- **AI встроен в карточки**, а не отдельный чат: вероятность сделки, риски, next action, черновик сообщения.
- **Проверка контрагента встроена** (блок counterparty-check / inn-bot): риск партнёра по ИНН прямо в карточке компании — арбитраж, ФССП, статус ЕГРЮЛ, скоринг.

## Чего НЕ делаем (анти-Битрикс24)
Задачи на всю компанию, проекты, склад, HR, диск, wiki, корпоративный чат.

## Роль inn-bot
Существующий Telegram-бот @INNRussia_bot (проверка контрагента по ИНН) становится
**блоком** AI Sales OS. В продажах это даёт уникальное преимущество: менеджер видит
риск контрагента в момент работы со сделкой, а не в отдельном сервисе.

## Принципы
1. Простота важнее полноты функций.
2. Скорость: поиск и проверки — быстро (цель поиска по CRM < 100 мс).
3. Надёжность важнее «модной» архитектуры.
4. Каждый блок можно включать поэтапно.
