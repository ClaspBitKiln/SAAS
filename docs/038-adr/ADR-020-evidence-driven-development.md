# ADR-020 — Evidence Driven Development

**Статус:** Принято (2026-06-21).

## Решение
Любое утверждение («compiles», «tests pass», «swagger works», «repository works»)
должно иметь подтверждение:
```
command + output  ·  screenshot  ·  CI log
```
Без доказательства статус утверждения = **UNPROVEN**.

## Практика
- В `docs/BUILD_STATUS.md` рядом со статусом хранится ссылка/вставка доказательства
  (команда + вывод или ссылка на CI).
- В PR/коммите, заявляющем «green», прикладывается лог прогона.
- AI-ассистент не пишет «работает», если не приложил вывод команды.
