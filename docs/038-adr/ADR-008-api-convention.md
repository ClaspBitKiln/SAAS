# ADR-008 — API Convention

**Статус:** Принято (2026-06-21).

## Решение
- **REST first**, **OpenAPI first** (контракт ведущий).
- **No GraphQL.**
- Cursor генерирует контроллеры из контрактов/декораторов (`@nestjs/swagger`).
- Версионирование API через префикс (`/v1`). Единый формат ошибок и пагинации (shared).
- DTO валидируются `class-validator`; ответы — отдельные response-DTO.

## Связанные
ADR-010 (порядок генерации), `011`/`012` (API контекстов).
