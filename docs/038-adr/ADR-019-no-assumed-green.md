# ADR-019 — No Assumed Green

**Статус:** Принято (2026-06-21).

## Решение
Запрещается считать build, unit, integration или e2e зелёными, если они **реально не
выполнялись**. Допустимые статусы:
```
NOT_RUN · GREEN · RED
```
Статус **ASSUMED_GREEN запрещён.**

## Практика
- По умолчанию любой тест/сборка = `NOT_RUN`, пока нет факта запуска.
- `GREEN`/`RED` ставится только по реальному выводу команды.
- Текущий журнал — `docs/BUILD_STATUS.md`.
