# ADR-024 — No Manual Status

**Статус:** Принято (2026-06-21).

## Решение
Запрещено руками писать `DONE / GREEN / PASSED`. Статусы вычисляются автоматически из:
CI · reports · coverage. `BUILD_STATUS.md` — **производный** документ (обновляется
пайплайном/скриптом из результатов CI, а не редактируется вручную).

## Практика
- CI пишет сводку в job summary и обновляет/прикладывает отчёты.
- Человек/AI может писать только `NOT_RUN` (отсутствие данных) и описания; вердикты
  `GREEN/RED/DONE` ставит автоматика по артефактам.
