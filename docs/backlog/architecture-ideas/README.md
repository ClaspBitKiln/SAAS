# Architecture Ideas Backlog

Сюда складываются новые архитектурные идеи во время Architecture Freeze (ADR-015 §1).
Не менять архитектуру по ходу — фиксировать идею здесь.

Формат записи (один файл на идею или строка в таблице):

| Дата | Идея | Статус | Причина |
|------|------|--------|---------|
| 2026-06-21 | Пример: вынести AI в отдельный сервис | DEFER | после первой выручки / реальной нагрузки |

Статусы: **ADOPT** (взять, но обычно после разморозки) · **DEFER** (отложить) · **REJECT** (отклонить).
Разморозка — после завершения Platform → Auth → Users → RBAC.
