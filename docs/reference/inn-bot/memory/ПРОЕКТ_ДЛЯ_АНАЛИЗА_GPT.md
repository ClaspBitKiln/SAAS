# INN-bot (@INNRussia_bot) — полное описание проекта для анализа

> Документ для передачи во внешнюю модель (GPT) с целью получить рекомендации по развитию, монетизации, маркетингу и технике. Дата среза: 21.06.2026.

---

## 1. Что это за проект

**Telegram-бот @INNRussia_bot** — бесплатная проверка российских компаний и контрагентов по ИНН: арбитражные дела (КАД / kad.arbitr.ru), долги ФССП, статус в ЕГРЮЛ, оценка рисков и связи (директор, учредители). Ввод 10 или 12 цифр ИНН без регистрации.

- Канал: **@INNRussia** (контент, автопостинг).
- Сайт-дашборд: `https://web-production-45a33.up.railway.app/dashboard?key=admin2026`
- GitHub: `github.com/ClaspBitKiln/inn-bot` (ветка main).
- Хостинг: **Railway** (US West, 1 replica), деплой v2.3 — ACTIVE.

**Идея монетизации (отложена):** базовые проверки бесплатно навсегда; расширенные PDF-отчёты — платно; для управляющих — пилот 90 дней бесплатно.

---

## 2. Технический стек

- Python 3.12, **FastAPI** (web/API), **aiogram 3** (бот).
- **SQLite** на Railway Volume (`/data/watchlist.db`).
- Таблицы: users, watchlist, events, companies, api_cache, counters, allowed_users, user_attribution, premium, payments.
- Основной файл: `bot/simple_app.py` (~2300+ строк).

**Источники данных по ИНН (каскад):**
1. **Checko API** — основной (арбитраж + ФССП).
2. **kad-прокси на VPS** (headful Chromium + Xvfb) — резерв для kad.arbitr.
3. **DataNewton API** — дополнительный.
4. **ЕФРСБ REST API** — банкротство/АУ (ИНН+СРО, 76% покрытие).
5. Планируется **DaData** (бесплатно, 10k/сутки).

**Платежи:** интеграция **ЮKassa** реализована (`/pricing`, callback `pay:`, `/webhook/payment`), но не финализирована (нет `YUKASSA_SHOP_ID` в env). Freemium: `FREE_DAILY_LIMIT=5`.

**Защита:** rate-limit 10/мин, webhook-секрет, закрытые служебные эндпоинты.

## 7. Ограничения и проблемы

1. Почта `innrussia@mail.ru` заблокирована на отправку (антиспам 550) → рассылка только вручную.
2. Telegram-аккаунт владельца ограничен (прошлая массовая рассылка) → проверку присутствия номера в TG делать нельзя.
3. PDF-продукт не готов → монетизация отключена.
4. Засвеченные ключи — нужна ротация.
5. Конкуренты: @egrul_bot (бесплатный), ЧЕК ИНН (5₽/проверка), bi_do (4990₽). Козырь — арбитраж + kad.arbitr.
